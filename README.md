# motion_driving

PC 워크스페이스 만들고, src 폴더 만든후  
src 폴더에   
PC에서 패키지 만들기   

```
ros2 pkg create --build-type ament_python motion_bot
```

PC에 git 다운하고 압축풀기  

setup.py 수정하기. 

```
colcon build  
```
